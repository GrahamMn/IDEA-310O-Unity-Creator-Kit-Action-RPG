## Cloning the Github Repo

It is required to use Git Large File Storage (`git lfs`) in order to get the file:
'Assets/Creator Kit - Beginner Code/Art/Texture/Buildings/Buildings_Albedo.tif'

Git LFS Download: https://git-lfs.com/

```bash
git clone git@github.com:GrahamMn/IDEA-310O-Unity-Creator-Kit-Action-RPG.git
git lfs install
git lfs pull
```
